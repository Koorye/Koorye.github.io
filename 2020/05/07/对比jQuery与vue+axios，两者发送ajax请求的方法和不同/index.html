еҜ№жҜ”jQueryдёҺVue+axiosзҡ„Ajax

еҸ‘иЎЁдәҺ 2020-05-07 жӣҙж–°дәҺ 2020-06-17 еҲҶзұ»дәҺ еүҚз«Ҝ йҳ…иҜ»ж¬Ўж•°пјҡ
жң¬ж–Үеӯ—ж•°пјҡ 3.6k йҳ…иҜ»ж—¶й•ҝ ≈ 3 еҲҶй’ҹ

axiosе’ҢjQueryзҡ„ajaxжң¬иҙЁдёҠйғҪжҳҜеҜ№еҺҹз”ҹXHRзҡ„е°ҒиЈ…пјҢдёҚиҝҮaxiosз¬ҰеҗҲжңҖж–°зҡ„ESи§„иҢғгҖӮеӣ жӯӨпјҢеңЁз”Ёvueзј–еҶҷjsж—¶пјҢдҪҝз”ЁaxiosиҖҢдёҚжҳҜjQueryзҡ„ajaxеә”иҜҘжҳҜжӣҙеҘҪзҡ„ж–№жЎҲгҖӮ

axiosе’ҢjQueryзҡ„ajaxз”Ёжі•еӨ§еӨҡзӣёеҗҢпјҢдёҚиҝҮиҝҳжҳҜжңүе°‘ж•°ең°ж–№дёҚдёҖж ·пјҢжҲ‘д»¬жқҘеҜ№жҜ”дёҖдёӢгҖӮз”ұдәҺaxiosз»ҸеёёдёҺvueз»“еҗҲпјҢиҝҷйҮҢз¬”иҖ…йҮҮз”Ёvue+axiosе®һзҺ°axiosзүҲajaxгҖӮ

postиҜ·жұӮ

еҮҶеӨҮжҺ§еҲ¶еҷЁ

йҰ–е…Ҳе®һзҺ°дёҖдёӘеҗҺеҸ°жҺ§еҲ¶еҷЁпјҢз¬”иҖ…жӯӨеӨ„йҮҮз”Ёspring bootпјҢеҸҜд»ҘзңӢеҲ°пјҢиҝҷдёӘжҺ§еҲ¶еҷЁйңҖиҰҒдёҖдёӘиҜ·жұӮеҸӮж•°idпјҢз„¶еҗҺдјҡиҝ”еӣһдёҖдёӘеҢ…еҗ«idе’ҢmsgдҝЎжҒҜзҡ„jsonпјҡ

еҮҶеӨҮhtml

жҲ‘д»¬жқҘеҮҶеӨҮдёӨдёӘжҢүй’®пјҡ

  • jQueryз»ҷжҢүй’®жҸҗдҫӣclickдәӢд»¶йңҖиҰҒз”ЁйҖүжӢ©еҷЁпјҢеӣ жӯӨиҝҷйҮҢжҸҗдҫӣдёҖдёӘid
  • vue+axiosеҸӘйңҖиҰҒдҪҝз”Ёv-on:clickжҲ–иҖ…з®ҖеҶҷ@clickе°ұеҸҜд»ҘжҢҮе®ҡдәӢд»¶

е®һзҺ°ajaxиҜ·жұӮ

иҜ·е…іжіЁд»ҘдёӢдёҚеҗҢпјҡ

  • еҸ‘йҖҒеҸӮж•°зҡ„ж–№ејҸ
  • жҺҘж”¶еҸӮж•°зҡ„ж–№ејҸ

vue+axiosзүҲajax

жӯӨеӨ„йңҖиҰҒеј•е…ҘQsжқҘиҪ¬жҚўж•°жҚ®зұ»еһӢпјҡ

jQueryзүҲajax

еҸ‘йҖҒеҸӮж•°зҡ„дёҚеҗҢ

дј з»ҹзҡ„jqueryжҠҠиҜ·жұӮж•°жҚ®е°ҒиЈ…еңЁdataдёӯпјҢdataжҳҜjsonзҡ„еҪўејҸпјҢйңҖиҰҒз”Ёеј•еҸ·иЎЁзӨәжҲҗе‘ҳеҗҚпјӣ

иҖҢaxios+qsзҡ„еҪўејҸдёҚйңҖиҰҒдҪҝз”Ёеј•еҸ·пјҢеҶҚйҖҡиҝҮQs.stringify()жҠҠж•°жҚ®иҪ¬жҚўжҲҗjsonгҖӮ

жҲ‘д»¬жқҘзңӢдёҖдёӢaxiosдёҚдҪҝз”ЁQs.stringify()дјҡеҸ‘з”ҹд»Җд№Ҳпјҡ

жҠҘдәҶ500й”ҷиҜҜпјҹеҶҚжқҘзңӢдёҖдёӢеҸ‘йҖҒзҡ„еҸӮж•°пјҡ

Request PayloadпјҹйӮЈд№ҲжӯЈеёёзҡ„еҸӮж•°жҳҜд»Җд№Ҳе‘ўпјҢдёӢйқўжҳҜjQueryеҸ‘йҖҒзҡ„еҸӮж•°пјҡ

еҸҜд»ҘзңӢеҲ°пјҢжӯЈеёёеҸ‘йҖҒзҡ„еҸӮж•°еә”иҜҘжҳҜForm DataпјҢиҝҷж ·еҗҺеҸ°жүҚиғҪжҺҘж”¶еҲ°дёҖдёӘjsonпјӣеҰӮжһңжҳҜRequest
PayloadпјҢеҗҺеҸ°иҝҳйңҖиҰҒйўқеӨ–зҡ„и§ЈжһҗжүҚиғҪеҫ—еҲ°е…¶дёӯзҡ„jsonгҖӮ

жүҖд»ҘпјҢ
jQueryеҸҜд»ҘзӣҙжҺҘеңЁdataдёӯзј–еҶҷjsonжқҘеҸ‘йҖҒpostиҜ·жұӮпјӣиҖҢaxiosйңҖиҰҒжҠҠзј–еҶҷзҡ„еҶ…е®№йҖҡиҝҮQsиҪ¬жҚўжҲҗjsonеӯ—з¬ҰдёІжүҚиғҪеҸ‘йҖҒpostиҜ·жұӮгҖӮ

жҺҘж”¶еҸӮж•°зҡ„дёҚеҗҢ

и§ӮеҜҹдёҠйқўзҡ„и„ҡжң¬жҲ‘д»¬еҸ‘зҺ°пјҢjQueryеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮresp.idгҖҒresp.msgеҫ—еҲ°jsonдёӯзҡ„ж•°жҚ®пјҢaxiosзҡ„ж•°жҚ®еҚҙеӯҳеңЁresp.dataдёӯгҖӮ

жҲ‘д»¬жқҘиҜ•дёҖдёӢжҠҠеҫ—еҲ°зҡ„jsonиҪ¬жҚўжҲҗеӯ—з¬ҰдёІпјҢдҪҝз”ЁJSON.stringify(resp)иҪ¬жҚўеҶҚз”Ёalert(JSON.stringify(resp))иҫ“еҮәпјҡ

jQueryзҡ„ajaxиҝ”еӣһзҡ„jsonпјҡ


axiosиҝ”еӣһзҡ„json:

еҸҜд»ҘзңӢеҲ°jQueryиҝ”еӣһзҡ„jsonдёӯеҸӘжңүж•°жҚ®пјӣaxiosеҚҙиҝ”еӣһдәҶеҗ„з§ҚдҝЎжҒҜпјҢеҰӮзҠ¶жҖҒз ҒгҖҒж—¶й—ҙгҖҒurlгҖҒиҜ·жұӮж–№жі•зӯүпјҢе…¶ж•°жҚ®еӯҳеңЁdataжҲҗе‘ҳдёӯпјҢиҝҷе°ұжҳҜжҲ‘д»¬иҺ·еҸ–ж•°жҚ®ж–№жі•дёҚеҗҢзҡ„еҺҹеӣ гҖӮ

getиҜ·жұӮ

ж—ўз„¶иҜ·жұӮж–№жі•еҸӘйңҖиҰҒжҢҮе®ҡmethodпјҢйӮЈд№ҲпјҢеҰӮжһңжҲ‘д»¬зӣҙжҺҘжҠҠдёӨдёӘи„ҡжң¬дёӯзҡ„methodж”№жҲҗgetпјҢиғҪеҗҰжӯЈеёёдҪҝз”Ёе‘ўпјҹ

jQueryжӯЈеёёиҝҗиЎҢпјҡ

axiosжҠҘ500зҠ¶жҖҒз Ғй”ҷиҜҜпјҡ

жЈҖжҹҘиҜ·жұӮеӨҙпјҢ жҲ‘д»¬еҸ‘зҺ°axiosжІЎжңүдј е…Ҙд»»дҪ•дҝЎжҒҜпјҢз”ҡиҮіеңЁеҺ»жҺүQs.stringifyе’Ңз»ҷidеҠ дёҠеј•еҸ·д№ӢеҗҺиҝҳжҳҜдёҖж · пјҡ

иҝҷиҜҙжҳҺaxiosдёҚиғҪйҖҡиҝҮdataз»ҷgetиҜ·жұӮжҸҗдҫӣеҸӮж•°гҖӮ

getе’Ңpostзҡ„е·®еҲ«

дәӢе®һдёҠпјҢgetиҜ·жұӮйҖҡиҝҮеңЁurlеҗҺзӣҙжҺҘи·ҹдёҠеҸӮж•°жқҘеҸ‘йҖҒиҜ·жұӮпјҢдҫӢеҰӮlocalhost:8080/login/?username=koorye&password=123пјҢе°ұжҳҜз»ҷеҗҺеҸ°еҸ‘йҖҒдәҶusernameдёәkooryeпјҢpasswordдёә123зҡ„еҸӮж•°гҖӮ

иҖҢpostеҲҷжҳҜдҪҝз”Ёrequest bodyдј йҖ’еҸӮж•°пјҢеҸӮж•°дёҚеӯҳе…ҘurlдёӯпјҢиҖҢжҳҜд»ҘйўқеӨ–зҡ„й”®еҖјеҜ№зҡ„ж–№ејҸдј йҖ’гҖӮ

jQueryдёҺaxiosзҡ„dataеңЁеҸ‘йҖҒgetиҜ·жұӮж—¶зҡ„дёҚеҗҢ

и§ӮеҜҹдёҖдёӢдёҠж–ҮдёӯjQueryе’ҢaxiosеҸ‘йҖҒgetиҜ·жұӮж—¶зҡ„urlпјҡ

jQueryдјҡж №жҚ®иҜ·жұӮзұ»еһӢиҮӘеҠЁеҲӨж–ӯжҠҠdataж”ҫеҲ°urlдёӯиҝҳжҳҜrequest bodyдёӯ пјҢжӯӨеӨ„jQueryеҸ‘йҖҒзҡ„getиҜ·жұӮзҡ„urlе°ұеёҰдёҠдәҶеҸӮж•°пјҡ

иҖҢaxiosе°ұдёҚиЎҢпјҢaxiosзҡ„dataеҸӘз”ЁдәҺз»ҷrequest bodyжҸҗдҫӣеҸӮж•°пјҢиҖҢдёҚиғҪеңЁurlдёӯйҷ„еҠ еҸӮж•°пјҡ

axiosзҡ„getиҜ·жұӮж–№ејҸ

axiosеҸ‘йҖҒgetиҜ·жұӮж—¶дёҚиғҪдҪҝз”ЁdataпјҢиҖҢиҰҒдҪҝз”ЁaxiosжҸҗдҫӣзҡ„дёҖз§Қж–°зҡ„зұ»еһӢparamsпјҢparamsе…¶дёӯзҡ„еҸӮж•°дјҡиў«зӣҙжҺҘеҠ еҲ°urlеҗҺпјҢжҲ‘д»¬жқҘиҜ•дёҖдёӢпјҡ

жҲҗеҠҹеҸ‘йҖҒиҜ·жұӮе№¶жҺҘж”¶ж•°жҚ®пјҡ

иҜ·жұӮеӨҙзҡ„urlдёӯжҲҗеҠҹйҷ„дёҠеҸӮж•°пјҡ

жҖ»з»“

еҲ°иҝҷйҮҢпјҢжҲ‘д»¬е·Із»ҸжҲҗеҠҹдҪҝз”ЁjQueryе’Ңaxiosзҡ„ajaxеҗ„иҮӘе®ҢжҲҗдәҶgetе’ҢpostиҜ·жұӮзҡ„еҸӮж•°еҸ‘йҖҒе’Ңз»“жһңиҺ·еҸ–пјҢжҲ‘д»¬жқҘжҖ»з»“дёҖдёӢдёҚеҗҢд№ӢеӨ„пјҡ

е…ідәҺеҸ‘йҖҒиҜ·жұӮ:

  • jQueryж— и®әжҳҜgetиҝҳжҳҜpostпјҢйғҪдҪҝз”ЁdataеҸ‘йҖҒж•°жҚ®пјҢjQueryдјҡиҮӘеҠЁеҲӨж–ӯиҜ·жұӮзұ»еһӢе№¶еҢ№й…Қ
  • axiosеңЁgetиҜ·жұӮж—¶дҪҝз”ЁparamsпјӣеңЁpostиҜ·жұӮж—¶дҪҝз”ЁdataпјҢе№¶дё”иҰҒжіЁж„ҸжҠҠеҸ‘йҖҒзҡ„dataйҖҡиҝҮQs.stringify()иҪ¬жҚўжҲҗjson

е…ідәҺжҺҘж”¶ж•°жҚ®:

  • jQueryжҺҘж”¶зҡ„ж•°жҚ®зӣҙжҺҘд»Ҙjsonзҡ„еҪўејҸеӯҳеңЁиҝ”еӣһзҡ„responseдёӯ
